## STS2 Relic Mod
### Self-made game mod for fun
- Add any relic(s) at the start of a run
- No UI required — fully config-driven
- Supports multiple relics
- Supports both newline and comma-separated formats

```
Slay the Spire 2\mods\
  StarterRelic.dll
  StarterRelic.json
  StarterRelic\
    config.txt
```
